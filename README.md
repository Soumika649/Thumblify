# 🎨 Thumblify – AI-Powered Thumbnail Generation Platform

**Thumblify** is an AI-powered web application that helps content creators generate **eye-catching, high-quality thumbnails** quickly and effortlessly.  
It uses the **Hugging Face Inference API** to generate thumbnails from text prompts and is built with **TypeScript** for better reliability and maintainability.

Thumblify is designed for creators on **YouTube, blogs, and social media**, enabling fast thumbnail creation without advanced design skills.

---

## 🌟 Why Thumblify?

Creating attractive thumbnails manually is time-consuming and requires design expertise.  
**Thumblify automates thumbnail creation using AI**, allowing users to generate professional thumbnails in seconds.

With Thumblify, users can:
- Generate thumbnails using AI prompts
- Iterate designs quickly
- Save time and improve content engagement

### 🖼️ AI Thumbnail Generation
